# Visualizzazione storico delle segnalazioni

Complessità: Bassa
ID: 4
Priorità: Bassa
Requisiti SOTA: Dashboard (../Requisiti%20che%20si%20vogliono%20riutilizzare/Dashboard%2019a1100dcfbd81a7bc72c0581e5baa22.md)

# DESCRIZIONE

Il sistema deve memorizzare uno storico dell’ultimo mese per permettere agli operatori di analizzare i dati passati. Questo può essere utile per identificare anomalie ricorrenti e migliorare la manutenzione del sistema. L’accesso a questi dati può essere fornito tramite un'interfaccia dashboard.
