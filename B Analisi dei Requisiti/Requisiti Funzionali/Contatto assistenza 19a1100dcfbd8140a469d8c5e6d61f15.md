# Contatto assistenza

Complessità: Bassa
ID: 1
Priorità: Media
Requisiti SOTA: Dashboard (../Requisiti%20che%20si%20vogliono%20riutilizzare/Dashboard%2019a1100dcfbd81a7bc72c0581e5baa22.md)

## Descrizione

In caso di problemi con il sistema come ad esempio guasti dei sensori, della centralina etc., il gestore deve poter contattare un centro assistenza tramite la dashboard o un’app mobile per poter soccorrere il conducente. Questo garantisce un supporto immediato e riduce il rischio di manomissioni.
